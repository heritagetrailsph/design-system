# Barabara

`Barabara-Regular.ttf` is the public Philippine tourism display typeface hosted by the
Department of Tourism campaign site.

- Source: https://www.itsmorefuninthephilippines.com/
- Original asset: `/static/media/BARABARAFINALRegular.7b70334e.ttf`
- Intended use in this project: title typography on the Barabara comparison study only

The face has an uppercase display character set, so the study uses uppercase title rendering and
keeps Poppins for body copy and interface text.
